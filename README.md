# Huffman Encoding Project
Huffman Encoding With C++

# How To Run
- g++ -std=c++11 *cpp -o Main
- You can just type "make" in terminal.
### Commands
- ./Main -i input_file -encode
- ./Main -i encoded_file -decode
- ./Main -s char
- ./Main -l
